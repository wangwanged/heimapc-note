什么是后加:? 

:是v-bind:的 一种缩写,v-bind绑定属性用的

如果你要求你的 属性是动态的 你就需要 :属性="变量"

name="张三" =>  var  name = '张三'

:name = "张三"  =>  var name = 张三  name等于一个叫张三的变量

:span = "12"  =>  var  span = 12  而不是 var  name = "12"
